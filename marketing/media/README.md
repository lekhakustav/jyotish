# Local media workspace (ignored)

This directory is a disposable local mirror and editing cache for media whose canonical copy is
in the verified Google Drive workspace. Put downloaded Veo renders, app recordings, audio,
editor caches, proxies, and exports here.

Do not rely on this directory as the only copy of an asset. Upload originals to Drive, verify the
Drive file ID, and record SHA-256/provenance in `marketing/registry/media-manifest.csv` before
clearing local files.
